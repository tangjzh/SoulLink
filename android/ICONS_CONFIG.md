# 图标配置说明

本应用使用Material Icons作为图标库。

## 安装和配置

```bash
npm install react-native-vector-icons
npx react-native link react-native-vector-icons
```

## 使用的主要图标

- `home` - 首页
- `psychology` - 数字人格  
- `chat` - 对话
- `favorite` - 情感匹配
- `plus` - 添加
- `send` - 发送
- `eye` / `eye-off` - 显示/隐藏

## 使用示例

```tsx
import Icon from 'react-native-vector-icons/MaterialIcons';

<Icon name="home" size={24} color="#6366F1" />
```
